## Why this course

This is the deepest connection to current teaching. Formal study of security at the systems level — beyond tools and certifications — is what enables research into how cybersecurity should be taught at faith-based institutions (Dissertation Topic B). Understanding threat models, cryptographic protocols, and formal security definitions grounds the curriculum argument in rigorous CS.

---

## Connection to Dissertation Topic B

Dissertation Topic B: *Cybersecurity curriculum for AG-affiliated institutions* — this course provides the technical depth to argue for what curriculum choices are academically defensible vs. merely vocational.
