# Research Papers

This folder contains Zotero BibTeX exports and the growing bibliography for the doctoral project.

## Files

- `bibliography.bib` — Monthly Zotero export (all sources, BibTeX format)
- `dissertation-sources.bib` — Subset of dissertation-relevant sources (added when dissertation topic is confirmed)

## Monthly export workflow

1. Open Zotero
2. Right-click your dissertation collection → **Export Collection**
3. Format: **Better BibTeX** (requires Better BibTeX plugin)
4. Save as `bibliography.bib` and replace the existing file in this folder
5. Commit: `[tools] Update Zotero BibTeX export - [Month YYYY]`

## BibTeX not installed yet?

Install the Better BibTeX plugin for Zotero:
[github.com/retorquere/zotero-better-bibtex/releases](https://github.com/retorquere/zotero-better-bibtex/releases)

Download the `.xpi` file → Zotero → Tools → Plugins → Install from File
