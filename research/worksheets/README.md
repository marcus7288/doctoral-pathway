# Research Worksheets

This folder contains completed Adler Analytical Reading Worksheets for every paper or book chapter read at Level 2 (Analytical) or Level 3 (Syntopical).

## Naming convention

```
[lastname]-[year]-[shorttitle].md
```

**Examples:**
```
guzdial-2015-learner-centered.md
margolis-2002-unlocking-clubhouse.md
nassi-1999-contextualization-theory.md
```

## Commit convention

After completing a worksheet:
```
[read] Adler worksheet - [lastname] [year] - [short title]
```

## Worksheet template

Use `docs/adler-reading-worksheet.md` as the template for every new worksheet.
