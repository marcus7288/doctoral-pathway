## How to Use This Repo

1. **OCW study** вЂ” Log every session in `ocw-portfolio/[course]/notes/session-log-YYYY-MM-DD.md`. Commit after every session using `[ocw]` prefix.
2. **Research reading** вЂ” Complete an Adler worksheet for every analytical read. File in `research/worksheets/`. Commit with `[read]` prefix.
3. **Reading logs** вЂ” Update `research/reading-logs/YYYY-MM.md` monthly. Commit with `[log]` prefix.
4. **Reading notes** вЂ” Copy `reading-notes/_template.md`. File as `[lastname-year-shorttitle].md`.
5. **Literature review** вЂ” Add each source to `literature-review/annotated-bibliography.md`.
